# ICE Deaths Research Assistant - Installation Instructions

## For Beta Testers

### Prerequisites
- Google Chrome browser (version 88 or higher)
- 5 minutes of your time

### Installation Steps

1. **Download the Extension**
   - Download and extract the extension zip file you received
   - OR clone the repository and navigate to the `extension` folder

2. **Enable Developer Mode**
   - Open Chrome and go to `chrome://extensions/`
   - Toggle **Developer mode** ON (switch in top-right corner)

3. **Load the Extension**
   - Click **Load unpacked** button (appears after enabling Developer mode)
   - Navigate to the extracted `extension` folder
   - Click **Select Folder**

4. **Verify Installation**
   - You should see "ICE Deaths Research Assistant" in your extensions list
   - The extension icon should appear in your Chrome toolbar
   - Click the icon to open the side panel

### First Use

1. **Log In**
   - Click the extension icon in your toolbar
   - Click "Sign in with Google"
   - Use the Google account you registered for beta testing

2. **Start Documenting**
   - Navigate to any news article or PDF about ICE incidents

### Troubleshooting

**Extension not loading?**
- Make sure you selected the `extension` folder, not a parent folder
- Check that all files (manifest.json, background.js, etc.) are present

**Can't sign in?**
- Make sure you're using the Google account registered for beta access
- Check your internet connection
- Try refreshing the extension: go to `chrome://extensions/` and click the refresh icon

**Quotes not saving?**
- Check your internet connection
- Make sure you're signed in (check the side panel)
- Look for error messages in the side panel

**Extension disappeared after Chrome restart?**
- This is normal for unpacked extensions
- Go to `chrome://extensions/` and click the refresh icon
- OR disable and re-enable the extension
